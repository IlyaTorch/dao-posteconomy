# Start backend server
```bash
cd backend
python uvicorn.py
```

# Install nvm v12.7.0
```bash
nvm use v12.7.0
```

# Deploy stamrt contracts to blockchain
```bash
truffle migrate
```

# Set Up Open Law env variables
```bash
export OPENLAW_EMAIL=your-email@gmail.com 
export OPENLAW_PASSWORD=your-password 
```

# Start client app
```bash
yarn start
```
